Detailed Description

The Connected Client API allows you to get and put documents, and listen for changes, directly on a remote database server (Sync Gateway or a Couchbase Lite sync listener), without any local database.

Gets the current revision of a document from the server.

You can set the unlessRevID parameter to avoid getting a redundant copy of a revision you already have.

Parameters
docIDThe document ID.
collectionIDThe name of the document's collection, or nullslice for default.
unlessRevIDIf non-null, and equal to the current server-side revision ID, the server will return error {WebSocketDomain, 304}.
asFleeceIf true, the response's body field is Fleece; if false, it's JSON.
callbackCallback for getting document.
contextClient value passed to getDocument callback
outErrorOn failure, the error info will be stored here.
