§ reduce

C4Slice(* C4ReduceFunction::reduce) (void *context)

Callback that returns reduced result as encoded data.

The data must remain valid until the next call to the reduce() callback! This function should also clear the internal accumulation state, in preparation for subsequent calls to the accumulate callback.
